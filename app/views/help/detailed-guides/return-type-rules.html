Part of:
How to format Environment Agency data

You must apply these specific rules to meet EA standards and submit data returns.

Return type names Field heading and rule
EPR/IED Air (point source) emissions return Unit: always use mg/m³ with Hourly for reference period. Measured as per M2 (or such other subsequent guidance as may be agreed in writing with the Environment Agency).
EPR/IED Air (ambient) monitoring return Unit: always use fibre/ml with 2 hour reference periods or g/hr for particulates.
EPR/IED Leachate level return Unit: always use monthly for operational cells, or quarterly for non-operational. State whether m>AOD or m>cell base as qualifier in text value field.
EPR/IED Leachate quality return Reference period: use MEPP, quarterly or annually. Units must be a concentration, physical parameter (pH, electrical conductivity, COD) or depth in m.
EPR/IED Surface water emissions return Unit: use either flow (l/s or m³/day) or concentration mg/l (exceptions: pH and temperature). Reference period: use either instantaneous or spot sample.
EPR/IED Groundwater quality/level return Unit: use either length (depth), concentration, or physical parameter such as pH, total alkalinity or electrical conductivity. Reference period: use MEPP, MEPP - up gradient, MEPP - down or cross gradient, or spot sample.

  • EPR/IED Landfill Gas Perimeter
  • EPR/IED Landfill Gas In Waste
  • EPR/IED Landfill Gas infrastructure monitoring
 Unit: use %v/v (exceptions - pressures mbr / millibar).
EPR/IED Sewer emissions return Reference period: use spot sample.
EPR/IED ETP or Tankering emissions return Reference period: use spot sample.
