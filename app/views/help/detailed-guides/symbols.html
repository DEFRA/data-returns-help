How to format Environment Agency data

Only use these approved symbols when you submit data to the Environment Agency. This will reduce ambiguity and improve consistency in your data returns.

Symbol Definition
% Percent, a number expressed as a fraction of 100
‰ Percentage per mille, a number expressed as a fraction of 1000
= Equal to
≠ Not equal to
<  Less than
<= Less than or equal to
>  Greater than
>= Greater than or equal to
~ Approximately equal to
+ Positive number (assume a number is positive if "+" symbol is not provided)
- Negative number
± Plus or minus, an indication of the error range associated with a value
∞ Infinity
* Asterisk or wildcard character
*.* Star-Dot-Star (used as a search criterion for everything containing a "." symbol)
.B~K Backup (file name extension)
| Pipe symbol (can be used to represent OR in a web search)
° Degree
2, 3 Squared or cubed
µ Micro
