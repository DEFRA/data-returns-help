Column description Column heading Description Example
EA reference
(mandatory) 		eaID This is a unique alphanumeric reference given in your Environment Agency permit or licence. It doesn’t include slashes, spaces or any other separators. AB12345Z1
Site name siteName This must be exactly the same as that given in your permit or licence. It’s not mandatory but it’s helpful to check against your EA reference. NJG Ltd
Return type
(mandatory) 		retType This is the type of return you’re sending, taken from a controlled list maintained and supplied by the Environment Agency. [not in permit or licence] Landfill Gas Monitoring
Start date dateSt Mandatory if there’s no period name. All dates and times have to be formatted YYYY-MM-DD (hyphens as separators). If you’re adding a time, use a space after the date entry then use the 24-hour clock in the format hh:mm:ss (colons as separators). See regime specific rules to find out if your time measurements must include seconds for spot sampling. 2013-08-05 15:28
End date dateEn All dates and times have to be formatted YYYY-MM-DD (hyphens as separators) hh:mm:ss (colons as separators). 2013-08-06 13:02
Monitoring period name periodName Mandatory if there’s no start date. This is given in regime specific rules. eg, January 2013 or Q1 2013
Sample reference sampleRef This is alphanumeric. It isn’t mandatory, so you can leave blank, but it helps to group together measurements, eg for water sampling. ABCD1234
Sample taken by sampledBy This is a free-text field identifying the person, group, contracting company or name of machinery responsible for taking the sample. For some returns this is a requirement (see regime specific rules), otherwise it can be left blank. This can’t include exotic characters or symbols. MRAF Ltd
Monitoring point monPoint This describes the sampling or emission point for gathering data, as agreed in the application, permit or working plan. For some returns this will have been given as ‘site’, if so leave blank. [specific location]
Substance (or sometimes measurement) parameter This is specified in a controlled list maintained and supplied by the Environment Agency. CH4
Numerical value value This is the numerical value of your data. It can include zeros or you can leave blank. See the regime specific rules to find out if you’re expected to provide decimal places. 123
Text value tValue You can use this field to give necessary text-based values - or Y/N or TRUE/FALSE. If it’s required this value will be specified in a controlled list maintained and supplied by the Environment Agency. FALSE
Unit for measurement Unit This applies to most returns but not all. For pH values leave blank. This is an alphanumeric field. Emissions readings must be given in %v/v or mg/m3. Temperatures must be given in DegC and pressure as KPa or mBar. mg/m3
Reference period MeasurePeriod Give this information if it’s a requirement of your permit or leave blank. 15 min average
Test methodology or standard testMethod Give this information if it’s a specified requirement of your permit or leave blank. BS EN1111
  Comments Free text
Commercial in confidence status CiC This is optional so you can leave blank or use Y/N, TRUE/FALSE. As the data return is for a public register it’s unlikely that you’ll need to use this field to label data as sensitive. YES
