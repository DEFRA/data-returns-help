Send your data returns online

How to format Environment Agency data

From:
Environment Agency
First published:
08 April 2016

Follow rules to format data that meets Environment Agency standards then send your data returns online.

Contents

  1. Environmental Permitting Regulations (EPR) data rules and return type rules
  2. Controlled lists

Before you can send your data returns file you need to:

  • apply data formatting rules for Environmental Permitting Regulations (EPR) and specific return types
  • save your data returns file as CSV

Then set up your data management systems to produce an automated export of data in the correct format.

Use the online service to submit landfill emissions compliance data covering up to 18 months in the past.

View an example data return.

Rules for Environmental Permitting Regulations data compliance

Follow rules about each field heading in your data return file to meet Environment Agency standards. See specific rules for EPR return types.

Controlled lists

You need to select the correct name from these controlled lists under certain field headings, such as parameter, when you complete your data returns. These lists support Environment Agency data standards and help reduce errors in your compliance monitoring.
